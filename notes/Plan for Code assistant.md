## proposal

I would like to create a code assistant based on chatGPT which helps with writing code. The program reads a script the user is working on and then finds functions that need to be generated based on docstrings. LLM's typically have a limited context window and therefore struggle to consider classes and functions made my the user that the LLM has need seen during training. Therefore, I would like to give context for these classes and functions based on what is imported in the script file.

Requirements
1. Script Selection: CLI interface to select which python script should be loaded by the network
2. program analysis: The program should detect all user-defined classes and functions in a given python script and its imported modules.
3. **Context Selection**: The user can specify the context for code generation. This might include a single function, a class, the uml outline of a class or a combination.
4. **Function/Method Generation**: The assistant should generate Python functions or methods based on the docstrings found in the code. These generated functions should adhere to the expected format and be customizable by the user.
5. **Version Control Integration**: Integrate with version control systems (e.g., Git) to allow users to commit and manage changes made by the assistant, making it easier to track, review, and revert changes if necessary.
6. Automated debugging. The assistant should also generate tests for the code it generates.
